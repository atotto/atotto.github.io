debian stretch GDM (gnome) 設定メモ

マウスカーソルのサイズを変更する $ gsettings set org.gnome.desktop.interface cursor-size 48 24,32,48,64で指定するらしい。 GUI(SettingsやTweak Tool)で変更できないような。 フォントサイズを変更する $ gsettings set org.gnome.desktop.interface text-scaling-factor 1.6 4Kディスプレイだと文字が小さくなるので、外部ディスプレイ利用時によく使う。
